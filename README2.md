# Social Media Posts App

A clean, minimalist micro-blogging and personal journaling dashboard built using Flutter. This application functions as a digital scratchpad to log daily tasks, development milestones, or quick thoughts within a high-contrast dark user interface.

## 🚀 Features

*   **Create Posts:** Add custom posts with distinct headers (Titles) and description paragraphs (Bodies).
*   **Live Feed Dashboard:** View your entries pinned sequentially in an easy-to-read card layout.
*   **State Management:** Powered by the `Provider` architecture for smooth, predictable data synchronization.
*   **Mock API Integration:** Uses the `http` package to communicate with the JSONPlaceholder `/posts` backend ecosystem to validate real-world CRUD operations.
*   **Instant Deletion:** Clean local removal of feed cards with zero background model errors.
